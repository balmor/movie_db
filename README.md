# Movie List React app

Frontend application using [The Movie Database API](https://developers.themoviedb.org). Built with React.

## Getting Started

Install packages:
```
yarn install
```

Use webpack-dev-server for local development:
```
yarn dev
```

Build the app:
```
yarn build
```
