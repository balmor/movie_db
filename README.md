# Movie List React app

Frontend application for display Movie Database using [The Movie Database API](https://developers.themoviedb.org) and reactjs.

## Getting Started

Install packages:

```
yarn install
```

Use webpack-dev-server for local development:

```
yarn start
```

Build the app:

```
yarn build
```
